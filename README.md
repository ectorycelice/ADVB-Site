# ADVB App 

Progressive Web App da Assembleia de Deus Vila Bocaina.

##  Tecnologias
- HTML5, CSS3, JavaScript
- Firebase Authentication
- Cloud Firestore
- Service Worker (PWA)

##  Funcionalidades
- Login com e-mail e Google
- Bíblia completa com busca
- Plano de leitura anual
- Programação de cultos
- Avisos e comunicados
- Painel administrativo
- Funciona offline

## Segurança
- Firebase Auth
- Regras Firestore
- Proteção XSS
- Content Security Policy
